## Funzione `from_json`

Carica una stringa formattata JSON

## Sintassi

from_json(_string_)

## Argomenti

* _string_ stringa json

## Esempi

* `from_json('{"qgis":"rocks"}') → { "qgis" : "rocks" }`
* `from_json('[1,2,3]') → [1,2,3]`
