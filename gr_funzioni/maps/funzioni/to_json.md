## Funzione `to_json`

Creare una stringa in formato JSON da una mappa, array o altro valore.

## Sintassi

to_json(_value_)

## Argomenti

* _value_ valore di input

## Esempi

* `to_json(map('qgis','rocks')) → {"qgis":"rocks"}`
* `to_json(array(1,2,3)) → [1,2,3]`
